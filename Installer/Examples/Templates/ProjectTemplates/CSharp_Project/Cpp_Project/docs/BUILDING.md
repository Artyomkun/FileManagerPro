# Сборка проекта

## Требования

### Системные требования

- Операционная система: Windows 10+, Linux, macOS
- Оперативная память: 4 GB минимум
- Свободное место: 1 GB

### Программное обеспечение

- C++ компилятор с поддержкой C++17:
  - GCC 7.0 или новее
  - Clang 5.0 или новее
  - MSVC 2017 или новее
- CMake 3.10 или новее
- Git (опционально, для клонирования)
