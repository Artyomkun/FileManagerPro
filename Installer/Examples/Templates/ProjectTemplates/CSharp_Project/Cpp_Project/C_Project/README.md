1. **Навигация**: Используйте дерево папок для быстрого доступа к файлам
2. **Редактирование**: Двойной клик по файлу открывает его для редактирования
3. **Поиск**: Ctrl+F для поиска по проекту
4. **Компиляция**: Настройте внешние инструменты для сборки
