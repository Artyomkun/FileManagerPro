## Как использовать с File Manager Pro

### Навигация

1. Откройте проект в File Manager Pro
2. Используйте дерево папок для навигации
3. Двойной клик открывает файлы для редактирования

### Поиск

- `Ctrl+F` - поиск в текущем файле
- `Ctrl+Shift+F` - поиск по всему проекту
- Фильтры по типу файлов (*.cpp,*.hpp)
