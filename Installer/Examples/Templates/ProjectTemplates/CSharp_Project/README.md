# File Manager Pro - C# Пример проекта

Этот проект демонстрирует возможности File Manager Pro при работе с C# проектами.

## Особенности

### 1. Современный Cшарп

- Использование .NET 6
- Новейший синтаксис C# 10
- Асинхронное программирование
- LINQ и расширенные возможности

### 2. Утилиты File Manager Pro

- Валидация имен файлов
- Работа с путями и расширениями
- Форматирование размеров файлов
- Анализ структуры проектов

### 3. Готовые примеры

- Классы для работы с файловой системой
- Демонстрация лучших практик
- Примеры использования утилит

## **Ключевые особенности:**

### **1. Современный C# 10/.NET 6:**
- **Новый синтаксис** с использованием `switch` выражений
- **Record types** готовы для использования
- **Top-level statements** для упрощения кода
- **Nullable reference types** для безопасности

### **2. Практические утилиты:**
- **Валидация имен** файлов для Windows
- **Определение типов** файлов по расширениям
- **Форматирование размеров** в читаемом виде
- **Работа с относительными** путями

### **3. Без побочных эффектов:**
- **Не создает файлы** на диске
- **Только анализ** и преобразование данных
- **Виртуальные структуры** для демонстрации

### **4. Демонстрация для File Manager Pro:**
- **Примеры интеграции** с .NET CLI
- **Готовые утилиты** для работы с файлами
- **Комментарии на русском** для понимания
- **Полная документация** в XML comments

Этот проект идеально подходит для демонстрации возможностей File Manager Pro при работе с C# проектами!
