# === Демонстрация навигации ===

Этот каталог содержит глубокую структуру папок  
для демонстрации возможностей навигации File Manager Pro.

Структура:

- DeepStructure/ - Глубокая вложенность папок
- symlinks_demo.lnk - Ярлыки для тестирования
- shortcuts.lnk - Еще один ярлык

Цели демонстрации:

1. Показать работу дерева папок
2. Демонстрация быстрой навигации
3. Тестирование отображения глубоких структур

Используйте:

- Стрелки навигации (← → ↑)
- Двойной клик для открытия
- Правая кнопка для контекстного меню
