# === Файл для демонстрации поиска===

Этот файл содержит ключевые слова для тестирования  
функции поиска в File Manager Pro.

## Ключевые слова

- filemanager
- поиск
- демонстрация
- пример
- тестирование

## Поисковые фразы

"File Manager Pro"
"кросс-языковый файловый менеджер"
"C C++ C# интеграция"

## Файл предназначен для

1. Тестирования поиска по содержимому
2. Демонстрации работы фильтров
3. Показа возможностей поиска с wildcards

## Примеры поисковых запросов

- *.txt - все текстовые файлы
- *manager* - файлы содержащие "manager"
- ??.txt - файлы с двумя символами перед .txt

- Используйте Ctrl+F для быстрого поиска!
