# === Примеры исходного кода для File Manager Pro ===

Этот каталог содержит примеры кода на разных языках программирования,
которые можно просматривать и редактировать с помощью File Manager Pro.

Содержимое:

1. C_Example/     - Пример проекта на языке C
2. Cpp_Example/   - Пример проекта на языке C++
3. CSharp_Example/ - Пример проекта на языке C#

Инструкция:

1. Откройте файлы двойным кликом для просмотра
2. Используйте поиск (Ctrl+F) для нахождения кода
3. Просматривайте структуру проекта в дереве папок

File Manager Pro поддерживает:

- Подсветку синтаксиса для разных языков
- Быстрый переход между файлами
- Поиск по содержимому файлов
