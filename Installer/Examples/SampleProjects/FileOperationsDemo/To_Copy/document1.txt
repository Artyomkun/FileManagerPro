# ===== Важный документ =====

Это пример текстового файла, который можно копировать  
с помощью File Manager Pro.

Содержимое документа:

1. Текстовые данные
2. Структурированная информация
3. Пример форматирования

Использование:

- Откройте файл двойным кликом
- Скопируйте с помощью Ctrl+C
- Вставьте в нужное место с помощью Ctrl+V

File Manager Pro поддерживает:

- Копирование файлов и папок
- Перетаскивание (Drag & Drop)
- Пакетные операции
