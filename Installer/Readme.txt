File Manager Pro v1.0.0
=======================

ОПИСАНИЕ
Профессиональный файловый менеджер с поддержкой
нативных C/C++ библиотек и современным интерфейсом.

ОСНОВНЫЕ ВОЗМОЖНОСТИ:
- Навигация по файловой системе
- Работа с файлами и папками
- Поиск и фильтрация
- Поддержка плагинов
- Многопоточные операции

СИСТЕМНЫЕ ТРЕБОВАНИЯ:
- Windows 10 или новее (64-bit)
- .NET Framework 4.8 или .NET 6.0
- 2 GB оперативной памяти
- 200 MB свободного места

УСТАНОВКА:
1. Запустите FileManagerPro_Setup.exe
2. Следуйте инструкциям мастера установки
3. Перезагрузите компьютер при необходимости
