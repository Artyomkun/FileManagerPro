Начало работы

File Manager Pro — это профессиональный файловый менеджер, разработанный на трёх языках программирования (C, C++, C#) с полной интеграцией.

Совет: Перед первым запуском убедитесь, что у вас установлены все необходимые компоненты.

Системные требования

  • Windows 10 или новее (64-bit)
  • .NET 6.0 Desktop Runtime
  • Visual C++ Redistributable 2015-2022
  • 2 GB оперативной памяти
  • 200 MB свободного места на диске

Установка

  1. Запустите FileManagerPro_Setup.exe
  2. Следуйте инструкциям мастера установки
  3. При необходимости перезагрузите компьютер
  4. Запустите программу из меню Пуск или с Рабочего стола

Навигация

Интерфейс программы

Главное окно состоит из следующих элементов:

  • Панель инструментов: Быстрый доступ к основным функциям
  • Дерево папок: Иерархическое представление файловой системы
  • Список файлов: Детальный вид содержимого текущей папки
  • Адресная строка: Отображение и редактирование текущего пути
  • Статус бар: Информация о выбранных файлах и состоянии диска
Совет: Используйте двойной клик по папке для быстрой навигации или по файлу для открытия.

Операции с файлами

Основные операции

  • Копирование: Ctrl+C или через контекстное меню
  • Вставка: Ctrl+V для вставки скопированных файлов
  • Удаление: Delete или Shift+Delete для полного удаления
  • Переименование: F2 или через контекстное меню
  • Создание папки: Ctrl+Shift+N или кнопка на панели инструментов

Расширенные операции

Пакетные операции

Выделите несколько файлов и выполните операцию сразу для всех

Фильтрация

Фильтруйте файлы по типу, размеру или дате изменения

Сортировка

Сортируйте по разным критериям кликом по заголовку колонки

Внимание: Операции удаления и перемещения требуют прав администратора для системных папок.

Интеграция и API

Архитектура взаимодействия

// C# вызывает C++ через P/Invoke [DllImport("LogicLib_Cpp.dll")] private static extern IntPtr CreateNavigator(); // C++ вызывает C функции напрямую extern "C" { #include "CoreLib_C/file_ops.h" int result = list_files(path, &files, &count); }

Поддерживаемые форматы интеграции

  • P/Invoke: Вызов неуправляемого кода из C#
  • COM Interop: Для интеграции с другими приложениями
  • Контекстное меню: Интеграция с проводником Windows
  • Ассоциации файлов: Открытие специальных форматов

Устранение неисправностей

  • Ошибка: "Не удалось загрузить DLL"

    Решение: Установите Visual C++ Redistributable 2015-2022 (x64).

  • Программа не запускается

    Решение: Установите .NET 6.0 Desktop Runtime.

  • Медленная работа с сетевой папкой

    Решение: Увеличьте таймауты в настройках или используйте кэширование.

  • Нет прав на операцию

    Решение: Запустите программу от имени администратора.

  • Контекстное меню не появляется

    Решение: Переустановите программу с опцией "Интеграция с проводником".

Журналы и диагностика

Журналы программы находятся в папке:

%LOCALAPPDATA%\FileManagerPro\Logs\

Для сбора диагностической информации выполните:

FileManager.exe --diagnostic --output=diagnostic.txt
