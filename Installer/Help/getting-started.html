Проверка системных требований

Перед установкой убедитесь, что ваша система соответствует требованиям:

Операционная система: Windows 10 или новее (64-bit)
Архитектура: Только x64 (64-bit)
.NET Runtime: .NET 6.0 Desktop Runtime
VC++ Redist: Visual C++ Redistributable 2015-2022
Оперативная память: 2 GB (рекомендуется 4 GB)
Место на диске: 200 MB свободного пространства
Дополнительно: SSD для лучшей производительности
Примечание: File Manager Pro использует трёхуровневую архитектуру (C/C++/C#), что требует установки всех компонентов.

Скачивание и подготовка

Выберите способ установки, подходящий для вас:

Стандартная установка

Рекомендуется для большинства пользователей:

  1. Скачайте установщик с официального сайта
  2. Запустите FileManagerPro_Setup.exe
  3. Следуйте инструкциям мастера установки

Тихая установка

Для системных администраторов и развёртывания:

FileManagerPro_Setup.exe /VERYSILENT /SUPPRESSMSGBOXES /NORESTART

Дополнительные параметры:

/DIR="C:\Program Files\FileManagerPro" – путь установки
/COMPONENTS="main,explorer" – выбор компонентов
/LOG="install.log" – запись лога

Разработчикам

Сборка из исходного кода:

git clone https://github.com/example/filemanager.git
cd filemanager
.\build.bat

Требуются: Git, Visual Studio 2022, .NET 6.0 SDK

Установка компонентов

Во время установки вам будут предложены следующие компоненты:

Мастер установки File Manager Pro

Основные компоненты:

  • Основные файлы программы: Исполняемые файлы и библиотеки
  • Интеграция с проводником: Контекстное меню Windows
  • Документация: Локальная справка и руководства
  • Примеры: Примеры использования и шаблоны
Важно! Для корректной работы требуется установить Visual C++ Redistributable. Если он не установлен, мастер установки предложит скачать его автоматически.
Совет: Рекомендуется установить все компоненты для полной функциональности.

Первый запуск и настройка

После успешной установки:

Окно первого запуска программы

Начальная настройка:

  1. Выберите язык интерфейса (русский/английский)
  2. Настройте горячие клавиши под свои предпочтения
  3. Выберите тему оформления (светлая/тёмная)
  4. Укажите папки для быстрого доступа
  5. Настройте параметры отображения файлов
Быстрый старт: Нажмите F1 в любой момент для вызова справки по текущему действию.

Проверка установки

Убедитесь, что всё работает корректно:

# Проверка установленных компонентов
FileManager.exe --diagnostic

# Тест C/C++ библиотек
FileManager.exe --test-native

# Создание отчёта о системе
FileManager.exe --system-report

Что должно работать:

  • Открытие программы из меню Пуск
  • Навигация по файловой системе
  • Контекстное меню в проводнике
  • Быстрый поиск файлов
  • Операции с файлами (копирование, удаление)
Диагностика: Если возникли проблемы, используйте команду --diagnostic для создания отчёта.

Часто задаваемые вопросы

Могу ли я установить программу на Windows 7?

Нет. File Manager Pro требует Windows 10 или новее из-за использования современных API .NET 6.0, которые не поддерживаются в Windows 7.

Почему требуются обе библиотеки: .NET и VC++?

Программа использует трёхуровневую архитектуру:

  • .NET 6.0 – для графического интерфейса (C#)
  • VC++ Redist – для логики навигации (C++)
  • WinAPI – для файловых операций (C)
Все три компонента необходимы для работы.

Как обновить программу до новой версии?

Доступно несколько способов:

  1. Автоматическое обновление через встроенный механизм
  2. Скачивание новой версии с официального сайта
  3. Через параметры программы (Настройки → Обновление)
Старая версия будет автоматически удалена перед установкой новой.

Можно ли перенести настройки на другой компьютер?

Да. Настройки хранятся в папке:

%APPDATA%\FileManagerPro\config.json
Скопируйте этот файл на другой компьютер для переноса настроек.
